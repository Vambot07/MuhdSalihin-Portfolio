6. HOT RELOAD (NO NEED FOR NODEMON)
================================================================================

NEXT.JS HAS BUILT-IN HOT RELOAD!
---------------------------------

When you run: npm run dev

Next.js automatically:
âœ… Watches for file changes
âœ… Recompiles changed files
âœ… Updates browser WITHOUT refreshing (Fast Refresh)
âœ… Preserves component state when possible

YOU DON'T NEED NODEMON!

How it works:
1. You edit a file and save
2. Next.js detects the change
3. Recompiles that file (takes 1-2 seconds)
4. Browser updates automatically
5. You see changes INSTANTLY

What triggers reload:
- Changing any .tsx/.ts file
- Changing .css files
- Adding new files
- Changing configuration files (requires restart)


WHEN TO RESTART SERVER:
-----------------------
You ONLY need to restart (Ctrl+C, then npm run dev) when you:
- Install new packages (npm install)
- Change next.config.ts
- Change tsconfig.json
- Change package.json


TURBOPACK:
----------
Next.js 16 uses Turbopack (new bundler)
- Faster than Webpack
- Faster hot reload
- Better performance

You'll see: "â–² Next.js 16.0.7 (Turbopack)"
This means Turbopack is enabled!

DEPLOYING TO VERCEL (RECOMMENDED):
-----------------------------------

STEP 1: Push to GitHub
1. Initialize Git (if not done):
   git init
   git add .
   git commit -m "Initial commit"

2. Create repository on GitHub.com
   - Go to github.com
   - Click "New repository"
   - Name it "muhdsalihin-portfolio"
   - Don't initialize with README

3. Push to GitHub:
   git remote add origin https://github.com/Vambot07/muhdsalihin-portfolio.git
   git branch -M main
   git push -u origin main


STEP 2: Deploy to Vercel
1. Go to vercel.com
2. Sign up/login with GitHub
3. Click "New Project"
4. Import your repository
5. Vercel auto-detects Next.js
6. Click "Deploy"
7. Wait 2-3 minutes
8. Your site is LIVE!


AUTOMATIC DEPLOYMENTS:
----------------------
After initial setup:
- Every git push â†’ Vercel redeploys automatically
- You get a preview URL for each branch
- Production = main branch


CUSTOM DOMAIN (OPTIONAL):
--------------------------
1. Buy domain (namecheap.com, godaddy.com)
2. In Vercel â†’ Settings â†’ Domains
3. Add your domain
4. Update DNS records (Vercel shows instructions)

ALTERNATIVE: Deploy to Netlify
-------------------------------

2. Connect GitHub repository

9. COMMON QUESTIONS & TROUBLESHOOTING
================================================================================

Q: Port 3000 is already in use?
A: Next.js will use port 3001 automatically
   OR kill the process: taskkill /IM node.exe /F

Q: Changes not showing?
A: Hard refresh browser: Ctrl + Shift + R

Q: Lock file error?
A: Delete lock: rm -rf .next/dev/lock

Q: Install new package?
A: npm install package-name
   Then restart server (Ctrl+C, npm run dev)

Q: Dark mode not working?
A: It follows system preferences
   Windows: Settings â†’ Personalization â†’ Colors â†’ Choose mode
   Mac: System Preferences â†’ General â†’ Appearance

Q: How to add new section?
A: 1. Create new component: components/NewSection.tsx
   2. Import in app/page.tsx
   3. Add to page: <NewSection />
   4. Add to navbar links

Q: Animations not working?
A: Check app/globals.css has animation definitions
   Make sure classes are applied correctly

Q: Build fails?
A: Run: npm run build
   Fix any TypeScript errors shown
